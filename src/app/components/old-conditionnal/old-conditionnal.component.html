Nouvelle structure

Aujourd'hui, pour gérer un affichage conditionnel, avec angular, on utilisera les directives {{"@if"}}, {{"@elseif"}} et {{"@else"}}.

Pour me tester, sélectionnez un des boutons suivants, sur les conditions {{`(>0 && <42)`}} / {{`(42 <=> 54)`}} / {{"AUTRE"}}.

@if(ifStateTestor){ @if(ifStateTestor > 0 && ifStateTestor < 42){

Je m'affiche uniquement quand la valeur de la propriété est supérieure à 0 et inférieure à 42.

} @else if(ifStateTestor >= 42 && ifStateTestor <= 54){

Je m'affiche uniquement quand la valeur de la propriété est comprise entre 42 et 54.

} @else{

Je m'affiche uniquement lorsque la valeur est inférieure à 0 ou supérieure à 54.

} }

Ancienne méthode

Avant, on utilisait la directive {{`*ngIf`}}, qui s'ajoutait en paramètre aux balises HTML (qui aujourd'hui, requiert également l'importation du CommonModule).

@if(ifStateTestor){

Je m'affiche uniquement quand la valeur de la propriété est supérieure à 0 et inférieure à 42.

Je m'affiche uniquement quand la valeur de la propriété est comprise entre 42 et 54.

Je m'affiche uniquement lorsque la valeur est inférieure à 0 ou supérieure à 54.
